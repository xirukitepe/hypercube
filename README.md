## Hypercube

#### Bundle the Gems:

```
$ bundle install
```

#### Running the tests:

> Tools used:
  * Spring
  * Rspec
  * Guard

#### Spring Status:

```
$ spring status
```

#### Enable Spring

```
$ rails g
```

Run specs with Spring

```
$ time spring rspec
```

#### Stop Spring

```  
$ spring stop
```

#### Run Guard

```
$ bundle exec guard
```

Please refer to the original documentations for more info!

* [Spring](https://github.com/rails/spring)
* [Guard](https://github.com/guard/guard-rspec)
